📅 Scheduled tasks: como criar

Você diz, em linguagem natural, "transforme isso em tarefa recorrente diária às 9h" e o Cowork cria o cron. Sem painel, sem código.

Prompt típico

Awesome, esse output ficou perfeito.

Transforme isso em uma scheduled task:
- Roda todo dia útil às 9h00 (horário de São Paulo)
- Output cai num arquivo markdown em ~/Cowork/OUTPUTS/trend-YYYY-MM-DD.md
- Se falhar, me manda email com o erro

Confirma o agendamento antes de criar.

💡 Dica

Sempre peça "confirma o agendamento antes de criar". Evita criar 3 cron jobs duplicados sem perceber.

📣 Onde a saída chega

Output sem destino bom é output perdido. Decida cedo onde vai cair: pasta, email, Slack, Notion, ou só ficar na conversa.

Padrão. Markdown ou PDF em OUTPUTS/. Você revisa quando quiser.

📧 Email diário

Resumo de 1 página chega no inbox. Bom pra coisas que você quer ler no café.

💬 Slack channel

#cowork-reports. Time todo vê. Bom pra accountability.

📝 Notion database

Vira histórico pesquisável. Bom pra trend reports, prospects, tasks.

🤝 Agent-to-agent: Claude ↔ Grok

Esse é o "truque mágico" que destrava dados sem API paga. Cowork abre o X.com, conversa com Grok como humano, traz a resposta e processa.

🔍 Como funciona

  1. Cowork abre Chrome → X.com (você já está logado)
  2. Navega até Grok (chat lateral do X)
  3. Posta sua pergunta como se fosse humano digitando
  4. Lê a resposta visualmente (computer use)
  5. Se for incompleta, faz follow-up
  6. Volta com texto estruturado pra você

⚡ Por que isso vale ouro

Grok tem firehose do X em tempo real. API X paga custa ~US$ 800/mês. Computer use + Grok = mesma intel, US$ 0 (fora token do Claude). É o que destrava trend scan.

🪢 Encadeamento de skills

Skills isoladas resolvem 1 tarefa. Skills encadeadas viram sistema. Pipeline diário típico:

A

Skill X Insight (9h)

Roda trend scan, devolve top 5 signals scored.

↓ output vira input ↓
B

Skill Content Angles (9h05)

Pega o #1 signal e gera 3 ângulos de vídeo no seu estilo.

↓ output vira input ↓
C

Skill Canva Slides (9h10)

Gera capa+slide template no Canva pra cada ângulo.

6

🚨 Erros silenciosos

Tarefa agendada que falha sem avisar é o pior cenário: token sobe, output some, você só descobre numa segunda-feira. Configure alertas desde o dia 1.

✓ Sinais saudáveis

  • ✓Email/Slack de confirmação a cada run
  • ✓Output sempre vem no mesmo lugar/formato
  • ✓Tempo de execução consistente
  • ✓Tokens previsíveis

✗ Sinais de problema

  • ✗Sumiço por 2-3 dias sem alerta
  • ✗Output quebrado/incompleto sem comentário
  • ✗Pico de tokens (provável loop)
  • ✗Conector deslogou silenciosamente

⚠️ Atenção

Toda scheduled task deve ter "heartbeat": uma mensagem de "estou viva" mesmo quando nada novo aconteceu. Sem isso, você não distingue "sem novidade" de "falhei".
